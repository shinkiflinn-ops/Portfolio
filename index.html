Analyse Technique

Système de Visée 8-Directions (Godot 4)

Implémentation d'un système de visée réactif pour le shooter 'Jojo Jambo'. La logique utilise des vecteurs normalisés pour gérer précisément les mouvements diagonaux.


func _update_aim_direction() -> void:
    var input_dir = Vector2.ZERO
    input_dir.x = Input.get_axis("ui_left", "ui_right")
    input_dir.y = Input.get_axis("ui_up", "ui_down")
    
    if input_dir == Vector2.ZERO:
        input_dir.x = -1 if animated_sprite.flip_h else 1
    
    aim_direction = input_dir.normalized()
    gun_tip.rotation = aim_direction.angle()

Logique : normalized() garantit que le mouvement diagonal n'est pas plus rapide que le mouvement horizontal/vertical (maintien de la longueur unitaire).
